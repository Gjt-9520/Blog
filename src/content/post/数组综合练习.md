## 数组常见操作

### 求最值

练习:已知数组元素为{33,5,22,44,55},找出最大值并打印出来

```
public class test {
    public static void main(String[] args) {
        int[] arr = {33, 5, 22, 44, 55};
        int temp = 0;
        for (int i = 0; i < 5; i++) {
            if (temp <= arr[i]) {
                temp = arr[i];
            }
        }
        System.out.println("数组中的最大值为:" + temp);
    }
}
```
