## IDEA

项目结构介绍

project(项目)包含了module(模块)包含了package(包)包含了class(类)

## 快速生成内方法

```
public class HelloWorld {
    //输入psvm后回车,自动生成内方法
    public static void main(String[] args) {
        //输入sout后回车,自动生成输出语句
        System.out.println();
    }
}
```

## 快捷键

ctrl + alt + L 自动格式化代码
